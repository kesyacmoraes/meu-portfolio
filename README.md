# 🚀 Meu Portfólio - Frontend & Cibersegurança

Olá! Eu sou a **Kesya**, estudante de programação focada em criar interfaces modernas, rápidas e, acima de tudo, seguras. Este projeto é o meu portfólio pessoal, onde partilho os meus projetos, formação académica e insights sobre o mundo da tecnologia.

## 🛠️ Tecnologias Utilizadas
* **Frontend:** HTML5, CSS3 (Design Responsivo & Estética Neon/Bento Grid) e JavaScript (ES6+).
* **Backend & Database:** Firebase (Firestore) para gestão de conteúdos dinâmica.
* **Segurança:** Implementação de Security Rules no Firebase para proteção de dados e integridade do banco de dados.
* **Hospedagem:** GitHub Pages.

## 🔒 Destaques de Segurança
Neste projeto, apliquei conceitos de segurança no desenvolvimento:
- **Sanitização de Dados:** Proteção básica no envio de formulários de contacto.
- **Firestore Security Rules:** Regras granulares que permitem leitura pública, mas restringem a escrita e eliminação de dados apenas a administradores.
- **HTTPS:** Comunicação segura garantida via SSL.

## 📂 Estrutura do Projeto
- `index.html`: Página principal com apresentação.
- `blog.html`: Feed de artigos dinâmico consumindo dados do Firebase.
- `skills.html`: Tabela de formação e competências técnicas/interpessoais.
- `js/`: Lógica de renderização e integração com Firebase.
- `css/`: Estilização personalizada com foco em UX/UI.
