# meu-portfolio
Desenvolvimento de Portfólio focado na experiência do utilizador (UX) e integridade de dados. Este projeto utiliza HTML5 e CSS3 para uma interface futurista com efeitos de iluminação neon, e JavaScript para dinamismo. O diferencial reside na integração com Firebase Firestore, permitindo um sistema de mensagens e comentários em tempo real (CRUD).
